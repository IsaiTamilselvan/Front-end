# Front-end Practice Repository

This repository is dedicated to practicing **JavaScript, HTML, and CSS** with a focus on **DOM manipulation** and building a strong front-end foundation.

## 🎯 Goals
- Strengthen core front-end skills
- Explore DOM APIs and event handling
- Practice dynamic UI updates and interactivity
- Build confidence in writing clean, maintainable code

## 📂 Contents
- **HTML & CSS**: Layouts, styling, and responsive design basics
- **JavaScript**: DOM manipulation, event listeners, and interactive components
- **Practice Projects**: Small exercises and mini-projects to reinforce concepts

## 🚀 Why this repo?
Front-end development is all about creating engaging user experiences. By consistently practicing DOM manipulation and combining it with solid HTML/CSS fundamentals, this repository helps establish a strong base for modern frameworks and libraries.
